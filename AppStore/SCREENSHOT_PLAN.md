# Flara Day Screenshot Plan

Goal: screenshots should show the real app experience, not marketing-only screens.

## Screenshot Set

Recommended first App Store screenshot set:

1. Create tab with recording interface
   - Show the main record/write entry experience.
   - Caption idea: Capture your day by voice.

2. Review draft screen
   - Show a cleaned journal draft with title/body.
   - Caption idea: Turn speech into a journal.

3. Calendar tab
   - Show saved entries on calendar.
   - Caption idea: Revisit entries by date.

4. Insights tab
   - Show memory card, current streak, entries this month, and month recap.
   - Caption idea: Notice patterns over time.

5. Settings tab
   - Show appearance, support, privacy, and connection.
   - Caption idea: Make reflection feel personal.

## Visual Rules

- Use Flara Day's final Aqua mint app icon.
- Use real UI screenshots with sample journal data.
- Do not show private real journal content.
- Use short, calm overlay captions if adding framed App Store screenshots.
- Capture in light mode first; optional dark mode screenshots can be added later.

## Required Capture Checklist

- iPhone 6.9-inch display screenshots for App Store Connect.
- iPhone 6.5-inch display screenshots if App Store Connect requires them for the selected device support.
- Keep status bar clean and consistent.
- Use sample entries with safe, non-sensitive content.

## Sample Journal Data

Use sample entries around:

- app testing
- evening walk
- creative work
- planning the week
- a quiet coffee break
- exercise
