# Flara Day Screenshot Plan

Last reviewed: July 16, 2026

The next distribution is TestFlight, so polished App Store screenshots are not
the immediate blocker. Capture the final set only after the beta UI stabilizes.

## Recommended Six-Screenshot Story

1. **Create** - idle recording screen or an active waveform without private text
   - Caption: `Capture your day by voice`
2. **Review** - multilingual cleaned draft with a safe sample title/body
   - Caption: `Keep the languages you speak`
3. **Calendar** - several sample entries across a month
   - Caption: `Revisit every day`
4. **Insights** - memory, streak, entries, and a full ten-theme Month Recap
   - Caption: `Notice what returns`
5. **Future Letter** - clean delivery card with In-App and Email choices
   - Caption: `Write to your future self`
6. **Appearance And Privacy** - Settings with theme/font/security controls
   - Caption: `Make it feel like yours`

## Apple File Requirements

- Provide between one and ten screenshots for the iPhone localization.
- Preferred current iPhone 6.9-inch portrait sizes include `1260 x 2736`,
  `1290 x 2796`, or `1320 x 2868` pixels.
- A 6.5-inch set is needed only when a 6.9-inch set is not supplied.
- Use `.png`, `.jpg`, or `.jpeg`.
- Recheck Apple's screenshot specification page at upload time.

## Capture Rules

- Use the Aqua mint icon and final production UI.
- Use fictional, non-sensitive journal and letter content.
- Do not show real email addresses, verification codes, or personal notifications.
- Keep the status bar time, battery, and connectivity visually consistent.
- Turn off AssistiveTouch and other floating accessibility controls before capture.
- Verify captions fit at every size and do not cover the actual UI.
- Capture light mode first; add dark mode only if it strengthens the story.

## Safe Sample Themes

- creative work
- evening walk
- exercise
- planning the week
- friendship
- sleep
- gratitude
- family time
- app testing
- quiet coffee break
